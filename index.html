Introduction

This portfolio highlights my work in the BAS Information Technology program. It will showcase a social media web application developed using Scrum methodology and demonstrates skills in backend development, frontend design, relational databases, web development, security practices, and project management.

Project Overview: Social Media Web Application

The main project included in this portfolio is a social media web application developed as part of a Scrum team project. The website allows users to create accounts, log in, and share posts with other users.

Backend Programming

The backend portion of the project handled the processing of user data and communication with the database. Server-side scripts were responsible for processing login requests, storing posts, and retrieving data for display. This helped me understand how servers process requests and interact with databases.

Security Practices

Input validation and exception handling were used to prevent invalid data from being processed. These practices help protect the application from unexpected errors and improve reliability.

Frontend Programming

The frontend of the social media website was developed using HTML and CSS. The interface allows users to interact with the application by creating posts, viewing content, and navigating through the site. Designing the user interface helped me understand how layout and usability affect the user experience.

Relational Database Processing

The application used a relational database to store user accounts and posts. MySQL was used to create tables and manage relationships between users and their content. Queries were used to insert new data and retrieve information from the database.

Security Practices

Database interactions were designed to ensure that only valid data was stored. Proper validation and careful query design help prevent issues such as data corruption or unauthorized access.

Web Development

This project demonstrates full-stack web development by combining frontend design, backend processing, and database integration. Developing the social media application helped me understand how multiple technologies work together to create functional web applications.

Networking and Security

Security considerations were important during the development process. Authentication mechanisms were implemented to ensure that only registered users could access certain features. Input validation and error handling help protect the application from invalid data and system crashes.

Scrum Project Management

The social media website was developed using the Scrum methodology. Scrum allowed the development team to organize tasks into manageable sprints and collaborate effectively throughout the project.

Scrum Practices Used

  • Sprint planning
  • Task backlogs
  • Team collaboration
  • Iterative development

Lessons Learned

Using Scrum helped improve communication within the team and allowed us to adapt to challenges during development. The project also highlighted the importance of clear task assignment and collaboration when working on software development projects.

Skills Developed

  • HTML and CSS web design
  • Backend programming concepts
  • MySQL database management
  • Web application development
  • Secure coding practices
  • Scrum teamwork and project management

Conclusion

This portfolio demonstrates the technical and collaborative skills developed during my studies in the BAS Information Technology program. Through the development of a social media web application, I gained experience working with web technologies, databases, and Scrum-based software development.
